# What the project consists of 

Create a simple Web Page that allows a user to:
    Add name of guests
    Logo of the guest company
    Date of the Meeting

This informations will be displayed like this:
    Main Company Logo
    Guest Company Logo
    Date of the Meeting
    Name of the guests 
    Greeting message to the guests
